Expands some common arguments for working with results data.

Used by mlos_viz as well.

Parameters:

  • exp_data (ExperimentData | None) – ExperimentData to operate on.

  • results_df (pandas.DataFrame | None) – Optional results_df argument. If not provided, defaults to ExperimentData.results_df property.

  • objectives (Optional[dict[str, Literal["min", "max"]]]) – Optional objectives set to operate on. If not provided, defaults to ExperimentData.objectives property.

Returns:

The results dataframe and the objectives columns in the dataframe, plus whether or not they are in ascending order.

Return type:

tuple[pandas.DataFrame, dict[str, bool]]
